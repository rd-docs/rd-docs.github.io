│  │  ├─ package.json
│  │  ├─ postcss.config.cjs
│  │  ├─ public # 静态资源
│  │  ├─ README.md
│  │  ├─ src
│  │  │  ├─ api # 接口请求

模板功能 ​

  • Vue3 + Vite4 + Typescript
  • Vant 组件库
  • Vue 全家桶(Pinia 状态管理、Vue-router 4)
  • Eslint、Stylelint、Commitlint 统一开发规范
  • husky + lint-staged
  • Axios 封装及接口管理
  • 开发环境支持 Mock 数据
  • 通用样式、Tailwindcss 原子类框架
  • 开发环境调试面板
  • vw 视口适配
  • 支持 SVG 图标自动注册组件
  • 业务组件、工具库
  • 支持主题切换模式
